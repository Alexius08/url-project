Was ist eine URL?
Der Begriff

URL(Uniform Resource Locater)

Die URL ist die Adresse, welche oben im Browser eingegeben wird, sie wird auch Internetadresse genannt.
Ein Beispiel für eine URL ist youtube.com
Die URL wird dann auf die gewünschte Seite bringen.
URLs werden jeweiles nur einmal vergeben, sie sind also eindeutig.

Was ist das DNS?
Der Begriff

DNS(Domain Name System)

Das DNS sorgt dafür, dass deine eingegebene URL in eine IP-Adresse umgewandelt wird.
Also wird www.example.com zu 192.0.2.44.
Diese schickt er zu deinem PC zurück und dein PC stellt dann eine Verbindung zu dieser IP her.
Der Server mit dieser IP, schickt dir dann die angeforderte Website, wenn du also youtube.com aufgerufen hast, wir dir das Video angezeigt.
Man kann sich das DNS also wie ein Telefonbuch vorstellen.
Du kennst den Namen und guckst nach unter welcher Nummer du die Person erreichen kannst.
Erst danach rufst du ihn unter dieser Nummer an.

Ein HTML Sprunglink
Der Begriff

Bei einem HTML Sprunglink kann zu einer gewissen Stelle im Text gesprungen werden.
Die Stelle zu der gesprungen wird, ist festgelegt durch eine ID.

Bsp:
< div id=“zielort“>Ziel < div >

Danach muss festgelegt werden von wo zu dieser Stelle gesprungen werden soll, also zum Beispiel von einem Inhaltsverzeichnis.
Dazu benutzt man den href und schreibt in die Anführungszeichen #id.
Für die id wird die id des Zielobjekts eingefügt.

<a href="#zielort">zum Anfang der Seite </a>

Wenn man im Body-Tag noch „scroll-behavior: smooth;“ einfügt, sorgt dies für einen scrollenden Übergand anstatt eines Sprunges.
